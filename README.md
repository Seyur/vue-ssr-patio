
# Vue SSR

## Build Setup

**Requires Node.js 7+**

``` bash
# install dependencies
npm install # or yarn

# serve in dev mode, with hot reload at localhost:8600
npm run dev

# build for production
npm run build

# serve in production mode
npm start
```
